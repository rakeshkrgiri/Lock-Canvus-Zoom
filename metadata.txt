description=The Lock Canvas Zoom plugin is designed to provide users with the ability to lock or unlock the zoom functionality on the QGIS map canvas. This is particularly useful when users want to maintain a fixed zoom level while interacting with the map. Upon installation, the plugin automatically adds a toolbar button that toggles the lock state of the canvas zoom.
